How to create and connect to ec2 insatnce?
AWS UI : Using Ec2 service in aws and connect through aws web browser | Through ssh using keys ssh -i privtekey username@pubipadd
AWS CLI : 
  Create an access key in aws profile -> security credentials -> secret access keys
  Download aws cli from web broswer
  In personal laptop terminal give -> aws configure , it'll ask to enter aws access keys -> configured
  You are connected to aws through cli, ex: list the current ec2 instances ec2 describe
  Example: https://boto3.amazonaws.com/v1/documentation/api/latest/reference/services/ec2/service-resource/create_instances.html

  using cft templates available in github(below reference) we can create the istances
